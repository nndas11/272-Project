# User Guides

## User Dashboard

### Accessing the Dashboard
- Click the profile icon in the top-right corner and select "View Profile".
- Navigate directly to `/profile`.

### Features
- **Account Information**: View and edit your full name and email address.
- **Account Balances**: View, add, and delete balances in multiple currencies (USD, EUR, GBP, JPY, CAD).
- **Trade History**: View your trade history and delete individual trades.

## Color System and Theming

The application uses a centralized theme system with support for both light and dark modes.

### Toggling Themes
- Click the sun/moon icon in the header to toggle between light and dark mode.
- Your preference is saved in your browser's local storage and will be remembered on your next visit.

### Color Palette
The color system is designed to be professional, consistent, and accessible.

- **Light Mode**: Optimized for light backgrounds with dark text for high contrast.
- **Dark Mode**: Optimized for dark backgrounds with light text to reduce eye strain.

Both themes use a primary blue for actions, green for success, red for danger, and orange for warnings. The shades are adjusted for optimal readability in each mode.
